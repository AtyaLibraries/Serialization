# Atya.Foundation.Serialization

Shared `System.Text.Json` configuration for foundation packages that need
consistent camelCase naming, string enums, null-ignore behavior, and
case-insensitive reads.

## Included APIs

- `JsonSerializerDefaults`
- `JsonSerializerConfiguration`
- `JsonSerializerConfigurationExtensions`

## Install

```bash
dotnet add package Atya.Foundation.Serialization
```
